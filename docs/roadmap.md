# Project Roadmap

## Phase 1: Scaffold and Architecture Alignment

- Monorepo structure established.
- Service boundaries documented.
- Infrastructure folders prepared.

## Phase 2: Baseline Implementations

- Frontend shell and API client setup.
- Core service bootstraps (User, Catalog, Streaming).
- Local docker-based development workflow.

## Phase 3: Platform Capabilities

- Event-driven communication via RabbitMQ.
- Observability stack integration.
- CI/CD pipeline and quality gates.

## Phase 4: Advanced Features

- Recommendation pipeline integration.
- Engagement jobs and autoscaling flow.
- Performance, resilience, and hardening.
